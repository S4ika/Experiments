Крестики-нолики 3х3 / 4х4 / 5х5
Сделано за обещание закрыть пролог, если в коде есть что то не нужное(на ваш взгляд), не переживайте все запланировано XD
Как играть?
Запускаешь проект -> Play -> Выбираешь размер поля (3х3 / 4х4 / 5х5) -> Обязательно Start game -> Наслаждаешься
Если хочется поменять поле то жмакаешь Change grid и выбираешь другое поле (Не забудь нажать start game, а то как играть, если игра не началась?!)=)
